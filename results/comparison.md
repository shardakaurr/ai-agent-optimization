# Baseline vs Optimized Comparison

## Task
Create a beginner-friendly user guide for creating a GitHub repository.

## Comparison Table

| Metric | Baseline (No Rules) | Optimized (Persona + AGENTS.md) |
|------|---------------------|----------------------------------|
| Structure | Poor / Inconsistent | Clear & Structured |
| Numbered Steps | Missing / Partial | Fully Present |
| Hallucinations | Present | Reduced / None |
| Readability | Low | High |
| Spec Compliance | Low | High |

## Observations
- Optimized output followed a clear structure.
- Hallucinations were reduced after applying rules.
- Instructions were more beginner-friendly.

## Conclusion
Using persona-driven prompts and AGENTS.md significantly improved reliability, structure, and quality of the output.

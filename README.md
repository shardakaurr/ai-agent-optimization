# AI Agent Optimization Project

This project investigates how AGENTS.md and MCP tools can improve reliability and quality of AI coding agents for content engineering tasks.

## Objectives
- Reduce hallucinations
- Improve instruction compliance
- Improve documentation quality
- Enable reproducibility

## Tools
- AGENTS.md
- MCP
- SpecKit
- GitHub

## Title

Beginner Guide: How to Create a GitHub Repository

## Overview

This guide explains how to create a new repository on GitHub using a web browser.  
It is written for beginners and uses simple, clear steps.

## Prerequisites

Before you start, make sure you have:

1.  A GitHub account
2.  A web browser (Chrome, Edge, Firefox, or Safari)
3.  An internet connection

If you do not have a GitHub account, you must create one first.

## Steps

### Step 1: Sign in to GitHub

1.  Open your web browser.
2.  Go to **<https://github.com>**.
3.  Click **Sign in**.
4.  Enter your GitHub username and password.

### Step 2: Start Creating a New Repository

1.  After signing in, look at the top-right corner.
2.  Click the **+** (plus) icon.
3.  Select **New repository** from the menu.

### Step 3: Enter Repository Details

1.  In **Repository name**, type a name (for example: `my-first-repo`).
2.  (Optional) Add a short description.
3.  Choose **Public** or **Private**:
    *   **Public**: anyone can see it
    *   **Private**: only you and selected people can see it

### Step 4: Initialize the Repository

1.  Check **Add a README file** (recommended for beginners).
2.  (Optional) Choose a `.gitignore` template.
3.  (Optional) Choose a license.

### Step 5: Create the Repository

1.  Scroll down.
2.  Click **Create repository**.

Your repository is now created.

## Validation

To confirm the repository was created successfully:

1.  You see a new page with your repository name at the top.
2.  The repository URL looks like:  
    `https://github.com/your-username/repository-name`
3.  A README file is visible if you selected one.

## Summary

You have learned how to:

*   Sign in to GitHub
*   Create a new repository
*   Set basic repository options
*   Verify that the repository exists

You are now ready to add files and start using GitHub.
